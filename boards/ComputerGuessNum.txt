notes: {
The classic "guess the number game" on an 8-bit computer.

To run this, switch to view mode (press enter) and hit r. Now think of a number between 0 and 127 inclusive, computer will try to guess it. Once the
computer makes a guess and the numpad says "ready", press 0 if the guess was too low, 1 if it was too high, or 2 if it is correct. Press e to enter
the input over the I/O port and then wait for the computer to make its move (unless the guess was correct, in which case the computer halts). The
only outcomes in this game are: 1) computer wins. 2) you cheated and computer indicates this with a -1. 3) you shut down or destroy the computer in
the process. The assembly code for this game is available at "computer stuff/developement/GuessTheNum.ncc".
}

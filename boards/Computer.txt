notes: {
Its a computer! It can run programs written in a very basic version of assembly and interface with external hardware through an I/O port.

Specifications: 8-bit Harvard architecture with reduced instruction set (RISC). Supports up to 256 bytes of RAM (only 16 bytes currently installed
though) and 512 bytes of program ROM (only 64 currently installed for 32 instructions). 16 registers are present in the CPU, but only 12 of these are
available for use as variables in programs.

Once a program is loaded into the ROM, the r button can be pressed to run it, p to pause execution, and h to halt. Due to the simplicity of this
thing (yes its actually very basic) programs take a decent amount of time to run as the CPU has a non-pipelined design. To speed up the computer a
lot, unlimited speed mode is recommended. As a final note, this design is quite dated now and not everything is subject to work properly. In a future
update to CircuitSim where a new set of logic rules are implemented, I plan on making a much better version of this thing that features cool stuff
like a memory cache, GPU, and instruction pipelining that this build doesn't have.
}
